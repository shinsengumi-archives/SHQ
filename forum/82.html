Suggestion about posting files to SHQ

by warg3791

I want to stress that this is just a SUGGESTION. We're not going to
play dictator with you or anything!

As you all know, we want the files to remain strictly text files.
That generally means that they're some sort of wordprocessor file.
The trouble I've run into with those is that they can occasionally be
hard to open. Also, they can involve extra work when it comes time to
transfer the info to the website.

The suggestion we would like to make is that if you have the "publish
to html" feature on your computer, please convert your files to this
first. This will make it easy to open and hopefully save Barb some
trouble transferring it later on. If you do not have this feature (I
think both MSword and WordPerfect shoud have it somewhere under File)
then feel free to send the info to me or another moderator who has
the feature. We can convert it and post it for you or send it back to
you via e-mail attachment so you can post it yourself. Whichever you
specify.

Again, we're not trying to pressure anyone. We're not going to start
deleteing any file that's not html or anything. It is merely a
suggestion. We simply want this to be as enjoyable and easy on
everyone involved as we can make it.

-MissBehavin.
