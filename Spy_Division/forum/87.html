How to post a new topic
1. To post a new topic, click on the New Topic button (may vary
depending on the community).
2. Once you have clicked the New Topic button it will take you to a
page where you enter your message and subject title. When you are
finished, you can choose to preview your topic by checking the preview
box next to the Add Post button

How to edit a post
Only the original author, administrator or moderator can edit a post
on a community.
1. To edit a message, click on the Edit link or image (usually a
checkmark).
2. Once you have clicked the Edit link it will take you to a page
where you can edit the message. You can choose to preview your reply
by checking the preview box next to the Edit Message button.

How to Bold Text Within a Post
Sometimes you'll want to emphasize a word or phrase within a post.
Making the text bold is often a great way to do this.
<strong> this is bold text </strong>
Result: this is bold text

How to Create a Link That Opens in a New Window
Linking somewhere else will take them away from a post. Adding the
target="_blank" parameter will open a new window so that they never leave.
<a href="http://www.ezboard.com/" target="blank"> this link will
pop-up a new window</a>
Result: this link will pop-up a new window
[if you notice on all the links at the top of the messageboard...I
made it so that a new page will be opened for everyone's convenience]
